## Tankad Tacos

# Purpose

A staff login system for a fictive late-night taco truck in Stockholm.

# Authentication

- Register user
- Log in user
- Create JWT
- Save/use JWT on frontend
- Protect at least one backend route
- Have at least one frontend page that only works when logged in

# Protected data

Protected admin dashboard:
- Restaurant name: Tankad Tacos
- Menu items: Tacos, Nachos, Quesadillas, Drinks
- Logged-in staff message

# Backend

Express
MongoDB
JWT

# Frontend

HTML & SCSS
JavaScript
Fetch API
